🎬 2D AR动画使用说明

⚠️ 重要提示

请确保您的手机和电脑在同一个WiFi网络下，这样才能正常访问AR应用。

📱 手机访问步骤

步骤1：获取访问地址

在手机浏览器中输入以下地址：

http://10.9.13.192:8080

或者点击下面的链接：

打开AR应用

步骤2：允许摄像头权限

当页面加载后，点击"测试AR功能"按钮，然后允许浏览器访问摄像头。

步骤3：准备AR标记

您需要以下任一方式显示AR标记：

  • 打印标记：将下面的图片打印出来
  • 屏幕显示：在另一台设备上显示标记图片
  • 手机分屏：在手机上同时显示AR应用和标记图片

步骤4：扫描标记

将手机摄像头对准AR标记图案，保持距离20-50cm，确保光线充足。

🎯 AR标记图案

请将摄像头对准此图案：

使用s1.png作为AR识别标记

🎬 动画效果

当成功检测到标记时，您将看到以下动画序列：

  1. frame1.png
  2. frame2.png
  3. frame3.png
  4. frame4.png

动画会循环播放，每帧间隔200毫秒。

🔧 故障排除

如果无法检测到标记：

  • 确保标记图案清晰可见
  • 保持充足的光线
  • 避免反光和阴影
  • 保持适当的距离（20-50cm）
  • 确保手机摄像头清洁

如果页面无法加载：

  • 确保手机和电脑在同一WiFi网络
  • 检查防火墙设置
  • 尝试使用不同的浏览器（推荐Chrome或Safari）

📞 技术支持

如果遇到问题，请检查：

  • 浏览器控制台的错误信息
  • 网络连接状态
  • 摄像头权限设置
